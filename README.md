## 关于
Bob 是一款 macOS 平台的 翻译 和 OCR 软件

本插件基于 vocabulary.com 提供的英文单词的英文释义


## 如何使用
1. 安装 Bob 软件
2. Release 中下载以 bobplugin 结尾的文件
3. 双击安装
4. 选择偏好的发音来源

## 编译打包
1. Bob文档 - [开发插件](https://bobtranslate.com/plugin/)
2. 根目录下执行 `node build.js` 自动打包为 bob 插件，打包路径为 `./release`
